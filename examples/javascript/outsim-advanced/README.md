# Node InSim - OutSim with Options (JavaScript + CommonJS)

## Requirements

- Node.js v18
- LFS

## Installation

```shell
npm install
```

## Connecting to LFS

Open `cfg.txt` in your LFS installation folder and edit the following lines:

```
OutSim Mode 1
OutSim Delay 10
OutSim IP 127.0.0.1
OutSim Port 30000
OutSim ID 1
OutSim Opts 1ff
```

Run the example app:

```shell
npm start
```
