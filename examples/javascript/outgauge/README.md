# Node InSim - OutGauge (JavaScript + CommonJS)

## Requirements

- Node.js v18
- LFS

## Installation

```shell
npm install
```

## Connecting to LFS

Open `cfg.txt` in your LFS installation folder and edit the following lines:

```
OutGauge Mode 1
OutGauge Delay 10
OutGauge IP 127.0.0.1
OutGauge Port 29998
OutGauge ID 0
```

Run the example app:

```shell
npm start
```
