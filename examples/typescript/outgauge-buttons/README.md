# Node InSim - OutGauge with InSim Buttons (TypeScript + ES Modules)

## Requirements

- Node.js v18
- LFS

## Installation

```shell
npm install
```

## Connecting to LFS

```shell
npm start
```

The app connects to `127.0.0.1:29999` by default.

## Production build

```shell
npm run build
npm run serve:production
```
