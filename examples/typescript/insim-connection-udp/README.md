# Node InSim - InSim connection using UDP (TypeScript + ES Modules)

## Requirements

- Node.js v18
- LFS

## Installation

```shell
npm install
```

## Connecting to LFS

Start LFS and type `/insim 29999` to open the InSim port.

Run the example app:

```shell
npm start
```

The app connects to `127.0.0.1:29999` by default.
