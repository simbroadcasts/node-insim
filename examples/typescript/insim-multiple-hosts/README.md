# Node InSim - InSim connection to multiple hosts (TypeScript + ES Modules)

## Requirements

- Node.js v18
- LFS

## Installation

```shell
npm install
```

## Connecting to LFS

```shell
npm start
```

The app connects to `127.0.0.1:29999` and `127.0.0.2:29998` by default.

## Production build

```shell
npm run build
npm run serve:production
```
